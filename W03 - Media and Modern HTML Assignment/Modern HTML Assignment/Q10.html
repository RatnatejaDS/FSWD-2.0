HTML vs. HTML5 Comparison

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 stands for HyperText Markup Language 5.
Introduced in the 1990s, it is the older version of the language. Introduced in 2014, it is the latest and most modern version of HTML.
Does not support native video and audio playback without plugins. Supports native audio and video elements (<audio> and <video>) for media playback without plugins.
